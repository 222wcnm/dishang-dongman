# 地上动漫 Perspective Skill

面向 Codex / Agent Skill 的“地上动漫”公开内容机制模型。

- 研究对象：Bilibili 创作者“地上动漫”
- 稳定账号：UID `175531853`
- 调研截止：2026-06-12
- 默认模式：内容机制与社区语境分析
- 角色模式：仅在用户明确要求模拟时启用

核心入口见 [`SKILL.md`](SKILL.md)，六维研究材料见
[`references/research/`](references/research/)。

## 安装

将本仓库克隆到 Agent 的 skills 目录：

```powershell
git clone https://github.com/222wcnm/dishang-dongman-perspective.git `
  "$HOME\.agents\skills\dishang-dongman-perspective"
```

## 边界

本项目只分析公开内容中可观察到的选题、表达、表演和社区互动机制，
不代表创作者本人或其真实私人观点。研究材料中的事实、角色表演、
社区解读与研究推断应分层理解。
