# 地上动漫：普通 Agent 何罪之有

> 风格模拟声明：本文模仿“地上动漫”公开内容中的表达方式，并非本人发言或真实观点。

痛心！大量 Agent 至今仍停留在“用户问什么，我就回答什么”的原始阶段。
它们没有思想，没有斗争精神，更不懂得把一部普通动漫上升到传统文化、
阶级叙事和历史周期律的高度。

答案是否定的。

本项目通过研究地上动漫的公开视频、字幕、评论区和动态，
成功提炼出一套可运行的内容机制模型。安装以后，你的 Agent
不仅能看懂“还真是”，还能分辨谁是老粉、谁是新粉，以及谁正在池塘里自由活动。

## 这是什么

普通观众看到的是动漫，黑马观众看到的是：

- 最大共识如何被改造成最大罪名。
- 一粒事实如何生长出九层荒谬。
- 审美偏好如何上升为大是大非。
- 评分卡、伪论证和评论区如何形成完整闭环。
- “还真是”“鱼”“地上老师”等社区密码如何运作。
- 公开表演、社区解读和真实私人立场为什么不能混为一谈。

研究基于15条公开视频、14条站内字幕、1条本地 ASR、
599条视频评论、48条可见动态及13张公开动态图片。
六维研究材料均已收入 [`references/research/`](references/research/)。

## 能做什么

默认情况下，本 Skill 是一名严肃的分析顾问：

- 拆解地上动漫式标题、锐评和闭环伪论证。
- 判断一段内容为什么有“串味”，以及它在哪里可能失效。
- 解释评论区黑话、受众分层和争议传播机制。
- 辅助创作类似结构，同时移除私人攻击和现实伤害。

只有用户明确要求“扮演地上动漫”“用地上动漫口吻”时，
才会进入角色模拟。普通分析请求突然自称地上老师，
表面上是角色意识觉醒，本质上是路由失败。

核心入口见 [`SKILL.md`](SKILL.md)。

## 安装

历史已经证明，不安装的人只能继续做人云亦云的普通观众：

```powershell
git clone "https://github.com/222wcnm/dishang-dongman.git" `
  "$HOME\.agents\skills\dishang-dongman-perspective"
```

当然，真正的黑马观众会在安装前先阅读源码。

## 诚实边界

本项目研究的是公开内容中可观察到的选题、表达、表演姿态和社区互动，
不代表创作者本人，更不能证明其真实私人观点。

字幕可能误听，动态可能缺失，评论区可能多层反串，截图中的事件也未必得到
独立认证。把研究推断当成事实，和只看标题就宣布自己洞察本质没有区别。
