- 单条播放中位数约25.3万，高于调研时约6.3万粉丝规模。
- 高评论密度样本：
  - `BV1PMq6YnEtf`：评论/播放约1.44%。
  - `BV1T1QuBNE3Y`：约1.23%。
  - `BV1PzqoYPEbh`：约1.13%。
  - `BV1f3z4BTE1s`：约1.13%。
- 高分享密度样本：
  - `BV1K8iUY5Ess`：分享/播放约2.90%。
  - `BV1PMq6YnEtf`：约2.27%。
  - `BV11pCYYEEvM`：约1.87%。

[研究推断] 内容的传播效率高度依赖“这也能这么说”的可转述性和评论参与，而不是单纯粉丝触达。置信度：高。

## 引流结构

用户给定关键词包含“引流狗”，但本次公开语料没有足够证据把它作为事实身份标签。

可观察到的引流机制是：

1. 选择已有强大粉丝群的动漫、角色、游戏或创作者。
2. 在标题直接触碰其核心共识。
3. 同时嵌入民族、阶级、性别、教育或AI等更大争议。
4. 让反对者、老粉和围观者都拥有评论理由。
5. 评论区再推荐下一个对象，形成连续选题池。

这应称为“争议型流量获取”，而不是未经证明地给作者定性。

## 动态先行的轻量引流

评分卡动态把完整引流结构压缩成低成本版本：

```text
借用番剧页面与高分共识
→ 用反常星级制造第一眼冲突
→ 用一句可截图、可转述的锐评提供传播钩子
→ 评论区争论评分密码并点名下一部作品
→ 高反应对象可继续扩写为视频
```

可见样本显示，评分卡具备独立互动能力：

- `1016109320049262595`《石纪元》：约3246赞、2384评。
- `1151828005935382535`《高木同学》：约3754赞、2000评。
- `1192646194013143106`《弹珠汽水瓶里的千岁同学》：约1661赞、698评。
- `1196380685256884231`《齐木楠雄的灾难》：约2506赞、478评。

[研究推断] 动态锐评既是内容本体，也是选题试验场：它用比视频更低的制作成本测试哪些共识倒置能引发高互动。现有数据没有逐条证明某动态必然转化为某视频，因此“试验场”置信度为中；“独立内容本体”置信度为高。

## 争议升级循环

```text
攻击共识对象
→ 字面观众反驳
→ 老粉把反驳定义为上钩
→ 评论区评选串法质量
→ 作者在后续视频引用反应
→ 争议本身成为下一条内容
```

`BV1K3zmY7E1A` 直接将私信威胁写入下一期叙事。

`BV1RQZ4ByEz1` 将“被开盒”及AI/宫崎骏争议转化成自我史诗化内容。这里只能确认作者公开视频中的说法，不能据此确认开盒事件的全部细节。

`BV1Nk516GEd9` 标题标注“补档”，评论 `rpid:302948812624` 询问为何删除，说明删除/补档也成为社区叙事的一部分；删除原因未核实。

## 液体喵事件：跨平台争议流水线

全站搜索筛出23条事件相关B站视频，调研时合计约85.9万播放。该数字只是关键词可见切片，不代表抖音直播和删除内容的完整传播量。

最大节点 `BV1foRjBSEvk`：

- 约40.8万播放。
- 约1.83万赞。
- 4404次分享。
- 标题同时包含“女友出轨”和“结婚证是假的”，一次性激活情感、真实性和人物反差三种点击理由。

### 流水线

```text
抖音直播抛出录音/聊天记录/关系指控
→ B站切片账号在数小时内发布“完整录音”“总览”“再次连麦”
→ 评论区用剧本、串、纯爱和被骗完成多版本解码
→ 和好或同框被包装成反转
→ 作者以“隐忍”和“二番战”重启事件
→ 录屏激励、福袋、抽奖和礼物接入直播
```

`BV1VS9XBMEci` 记录作者要求观众把关联账号艾特上麦，并主动给出“不是剧本/可起诉/有悬赏”的传播钩子。

`BV1jmER6eEwq` 中作者明确：

- 鼓励“不掐头去尾”的录屏与切片。
- 宣传“地上激励计划”。
- 让观众输入指定口令参与福袋。
- 将声称属于对方的私密物品包装成抽奖“周边”。

动态楼中楼 `rpid:302205104257` 又由作者本人确认“直播间抽”。

[研究推断] 事件形成了“直播生产原料—切片账号分发—评论区裁判—下一轮直播回收流量”的闭环。置信度：高。

### 炒作与剧本必须分开

- **炒作**：主动预告、公开连麦、悬赏、切片激励、福袋和二番战均有可核验行为证据。置信度：高。
- **纯剧本**：没有找到双方承认从头到尾合谋虚构全部事实的原始声明。置信度：低。
- **高度表演化**：双方真实关系素材被公开演出、剪辑和悬念化。置信度：高。

这三项不能合并成一句“既然炒作，所以全部是假的”。

## 风险与边界

- 争议对象从虚构角色扩展到真实个人或现实事件时，诽谤和骚扰风险显著上升。
- “开盒”只能作为作者公开叙事研究，不能收集或传播私人信息。
- 评论把内容称为剧本，不证明事件是编造。
- 作者发布截图不等于截图内容已经独立证实。
- 删除、补档和评论不可见都可能有平台、作者或其他原因，不能单因社区说法下结论。
- Skill在借鉴流量机制时，应保留“共识倒置”和“尺度错位”，移除现实伤害、隐私和未经证实的指控。
- 关系争议中的性名誉、健康、欠款、违法和报警说法均未独立认证。
- 反向号召观众“不要去问”仍可能实际引导骚扰。
- 把私密物品接入福袋或抽奖，会把现实伤害直接转化为互动和收益。
- 受众知道内容可能是串，不代表被指认的现实个人不会遭受长期后果。
